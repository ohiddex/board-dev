Question-2: What is the difference between innerHTML, innerText, and textContent ?

There can use innerHTML to change the HTML content (including tags) inside an element, innerText to change the visible text content inside an element while keeping CSS styles in mind, and textContent to change all of the text content inside an element, including hidden elements, without considering CSS styles.

Question-3: What is event delegation in the DOM?

Putting a single event listener to a parent element rather than each child element is known as event delegation.

Question-4: What is event bubbling in the DOM?

Event bubbling happens when an event on an element (such as a button click) "bubbles up" to its parent elements.
